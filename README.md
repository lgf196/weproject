# locally

> 本地生活小程序
> [博客](https:lgf196.top)
## 项目运行
 - 下载微信小程序的开发工具
 - 将这个项目下载下来导入微信开发工具中去
 - 找到微信开发工具中的详情这个按钮，点击它，然后将:不校验安全域名、TLS 版本以及 HTTPS 证书这个选项勾上
